## 知识概述

有理数的运算是初中代数的基础，包括加法、减法、乘法、除法、乘方以及混合运算。掌握运算的法则和运算律是关键。

## 核心概念

### 加法

**有理数加法法则**：
1. 同号两数相加，取相同的符号，并把绝对值相加
2. 异号两数相加，取绝对值较大的符号，用较大的绝对值减去较小的绝对值
3. 一个数同 0 相加，仍得这个数

**加法运算律**：
- 交换律：$a + b = b + a$
- 结合律：$(a + b) + c = a + (b + c)$

### 减法

**减法法则**：减去一个数，等于**加上这个数的相反数**。

$$
a - b = a + (-b)
$$

### 乘法

**乘法法则**：
1. 同号得正，异号得负，并把绝对值相乘
2. 任何数与 0 相乘，都得 0

**乘法运算律**：
- 交换律：$a \times b = b \times a$
- 结合律：$(a \times b) \times c = a \times (b \times c)$
- 分配律：$a \times (b + c) = a \times b + a \times c$

### 除法

**除法法则**：除以一个不等于 0 的数，等于**乘以这个数的倒数**。

$$
a \div b = a \times \frac{1}{b} \quad (b \neq 0)
$$

### 乘方

求 $n$ 个相同因数的积的运算叫做**乘方**，$a^n$ 中 $a$ 叫底数，$n$ 叫指数。

$$
a^n = \underbrace{a \times a \times \cdots \times a}_{n\text{个}}
$$

**特别地**：
- $(-a)^n$：当 $n$ 为偶数时为正，$n$ 为奇数时为负
- $-a^n = -(a^n)$，注意与 $(-a)^n$ 的区别

  title="例1：加减混合运算"
  problem="计算：$-20 + (-14) - (-18) - 13$"
  steps={[
    "先统一成加法：$-20 + (-14) + 18 + (-13)$",
    "同号相加：$[(-20) + (-14) + (-13)] + 18$",
    "$= -47 + 18$",
    "$= -29$"
  ]}
  answer="$-29$"
/>

<Example
  title="例2：乘除混合运算"
  problem="计算：$(-81) \div \frac{9}{4} \times \frac{4}{9} \div (-16)$"
  steps={[
    "先把除法转化为乘法：$(-81) \times \frac{4}{9} \times \frac{4}{9} \times (-\frac{1}{16})$",
    "负数的个数为 2（偶数），结果为正数",
    "先约分：$81 \times \frac{4}{9} = 36$，$36 \times \frac{4}{9} = 16$",
    "$16 \times \frac{1}{16} = 1$"
  ]}
  answer="$1$"
/>

<Example
  title="例3：乘方运算"
  problem="计算：$(-2)^3$ 和 $-2^3$，并说明区别"
  steps={[
    "$(-2)^3$：底数是 $-2$，即 $(-2) \times (-2) \times (-2) = -8$",
    "$-2^3$：底数是 $2$，即 $-(2 \times 2 \times 2) = -8$",
    "结果相同（巧合），但意义不同",
    "再比较：$(-2)^4 = 16$，而 $-2^4 = -16$"
  ]}
  answer="$(-2)^3 = -8$，$-2^3 = -8$；但 $(-2)^4 = 16 \neq -2^4 = -16$。注意底数的区别。"
/>

<Example
  title="例4：利用运算律简便计算"
  problem="计算：$(-\frac{1}{12} + \frac{2}{3} - \frac{3}{4}) \times (-24)$"
  steps={[
    "利用分配律：$(-\frac{1}{12}) \times (-24) + \frac{2}{3} \times (-24) - \frac{3}{4} \times (-24)$",
    "$= 2 + (-16) + 18$",
    "$= 2 - 16 + 18$",
    "$= 4$"
  ]}
  answer="$4$"
/>

## 公式定理

<Theorem title="有理数混合运算顺序">
  1. 先乘方，再乘除，后加减
  2. 同级运算从左到右进行
  3. 有括号的先算括号里面的（小括号 → 中括号 → 大括号）
</Theorem>

## 易错提醒

<CommonMistake title="运算顺序错误">
  混合运算最容易犯的错误就是运算顺序不对。例如：
  $$-2 + 3 \times 4 = -2 + 12 = 10 \quad (\text{正确})$$
  $$-2 + 3 \times 4 = 1 \times 4 = 4 \quad (\text{错误！先做了加法})$$
  
  记住口诀：**"先乘除，后加减"**
</CommonMistake>

<CommonMistake title="负号与减号的混淆">
  在式子 $-3 - 5$ 中，第一个 "$-$" 是负号（性质符号），第二个 "$-$" 是减号（运算符号）。
  
  正确理解：$-3 - 5 = (-3) + (-5) = -8$
</CommonMistake>

<CommonMistake title="分数运算时忘记通分">
  分数相加减时，必须先通分再计算。
  
  例如：$\frac{1}{2} + \frac{1}{3} = \frac{3}{6} + \frac{2}{6} = \frac{5}{6}$（正确）
  $\frac{1}{2} + \frac{1}{3} = \frac{2}{5}$（错误！不能分子分母分别相加）
</CommonMistake>
