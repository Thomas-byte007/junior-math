## 知识概述

**有理数**是初中数学的基石。它扩展了自然数的范围，引入了负数的概念，使得减法运算不再局限于"大减小"。有理数的学习从生活中的相反意义的量开始，逐步建立数轴、相反数、绝对值等核心概念。

## 核心概念

### 正数和负数

- **正数**：大于 0 的数，如 $+5$, $3.14$, $\frac{2}{3}$
- **负数**：在正数前面加上"$-$"号的数，如 $-5$, $-3.14$, $-\frac{2}{3}$
- **0**：既不是正数，也不是负数，是正负数的分界

### 有理数的定义

**有理数**：整数和分数统称为有理数。

$$
\text{有理数} = \begin{cases}
\text{整数} & \begin{cases}
\text{正整数: } 1, 2, 3, \ldots \\
0 \\
\text{负整数: } -1, -2, -3, \ldots
\end{cases} \\[6pt]
\text{分数} & \begin{cases}
\text{正分数: } \frac{1}{2}, \frac{3}{4}, 0.5 \\
\text{负分数: } -\frac{1}{2}, -\frac{3}{4}, -0.5
\end{cases}
\end{cases}
$$

> 注意：**有限小数**和**无限循环小数**都可以化为分数，因此它们也属于有理数。例如 $0.5 = \frac{1}{2}$，$0.\dot{3} = \frac{1}{3}$。

### 数轴

规定了**原点**、**正方向**和**单位长度**的直线叫做数轴。

```
    ────┼────┼────┼────┼────┼────┼────┼────▶
       -3   -2   -1    0    1    2    3
```

数轴的三要素：原点、正方向、单位长度，缺一不可。

### 相反数

只有**符号不同**的两个数互为相反数。$a$ 的相反数是 $-a$，特别地，$0$ 的相反数是 $0$。

**性质**：若 $a$ 与 $b$ 互为相反数，则 $a + b = 0$。

### 绝对值

数轴上表示数 $a$ 的点与原点的距离叫做 $a$ 的**绝对值**，记作 $|a|$。

$$
|a| = \begin{cases}
a, & a > 0 \\
0, & a = 0 \\
-a, & a < 0
\end{cases}
$$

**性质**：
- $|a| \geq 0$（绝对值非负性）
- $|-a| = |a|$
- 若 $|a| = |b|$，则 $a = b$ 或 $a = -b$

## 公式定理

<Theorem title="有理数加法法则">
  1. **同号两数相加**：取相同的符号，并把绝对值相加。
  
  2. **异号两数相加**：取绝对值较大的加数的符号，并用较大的绝对值减去较小的绝对值。互为相反数的两个数相加得 $0$。
  
  3. **一个数同 $0$ 相加**，仍得这个数。
</Theorem>

<Theorem title="有理数乘法法则">
  1. **同号得正，异号得负**，并把绝对值相乘。
  
  2. **任何数与 $0$ 相乘**，都得 $0$。
  
  3. 多个非零因数相乘，负因数的个数是**偶数**时积为正，是**奇数**时积为负。
</Theorem>

## 典型例题

<Example
  title="例1：求绝对值与比较大小"
  problem="求 $|-5|$、$|3|$、$|0|$ 的值，并比较 $-\frac{3}{4}$ 和 $-\frac{2}{3}$ 的大小。"
  steps={[
    "求绝对值：$|-5| = 5$（$-5$ 到原点的距离为 5），$|3| = 3$，$|0| = 0$",
    "比较 $-\frac{3}{4}$ 和 $-\frac{2}{3}$：先求绝对值 $|-\frac{3}{4}| = \frac{3}{4} = \frac{9}{12}$，$|-\frac{2}{3}| = \frac{2}{3} = \frac{8}{12}$",
    "因为 $\frac{9}{12} > \frac{8}{12}$，所以 $|-\frac{3}{4}| > |-\frac{2}{3}|$",
    "两个负数比较大小，绝对值大的反而小，所以 $-\frac{3}{4} < -\frac{2}{3}$"
  ]}
  answer="$|-5| = 5,\ |3| = 3,\ |0| = 0$；$-\frac{3}{4} < -\frac{2}{3}$"
/>

<Example
  title="例2：有理数混合运算"
  problem="计算：$(-2) + 3 \times (-4) - (-6) \div 2$"
  steps={[
    "先算乘除：$3 \times (-4) = -12$，$(-6) \div 2 = -3$",
    "原式变为：$(-2) + (-12) - (-3)$",
    "减去负数等于加上正数：$-2 - 12 + 3$",
    "从左到右计算：$-2 - 12 = -14$，$-14 + 3 = -11$"
  ]}
  answer="$-11$"
/>

<Example
  title="例3：数轴与绝对值综合"
  problem="已知 $|x - 2| = 3$，求 $x$ 的值。"
  steps={[
    "由 $|x - 2| = 3$ 可知，$x - 2$ 到原点的距离为 3",
    "所以 $x - 2 = 3$ 或 $x - 2 = -3$",
    "解 $x - 2 = 3$ 得 $x = 5$",
    "解 $x - 2 = -3$ 得 $x = -1$"
  ]}
  answer="$x = 5$ 或 $x = -1$"
/>

## 易错提醒

<CommonMistake title="误认为 $-a$ 一定是负数">
  $-a$ 不一定是负数。当 $a$ 本身是负数时，$-a$ 是正数。
  例如 $a = -3$，则 $-a = -(-3) = 3$ 是正数。$a$ 为任意有理数。
</CommonMistake>

<CommonMistake title="绝对值化简时忘记判断正负">
  在化简含有绝对值的表达式时，不能直接去掉绝对值符号，必须先判断绝对值内表达式的正负。
  
  例如化简 $|x - 2|$：
  - 当 $x \geq 2$ 时，$|x - 2| = x - 2$
  - 当 $x < 2$ 时，$|x - 2| = 2 - x$
</CommonMistake>

<CommonMistake title="有理数运算中的符号错误">
  在进行有理数混合运算时，最容易出错的是符号。
  
  记住以下口诀：**"同号得正，异号得负"**
  - $(-a) + (-b) = -(a + b)$
  - $(-a) - (-b) = -a + b$（减去一个负数等于加上正数）
  - $(-a) \times (-b) = ab$
</CommonMistake>
